# Business Intelligence Dashboard

Interactive Business Intelligence Dashboard built using Python, Plotly Dash, Pandas, and Machine Learning.

## Features

- KPI Analytics
- Revenue Forecasting
- Regional Performance Analysis
- Interactive Dashboard
- Data Cleaning & EDA
- Time-Series Analysis

## Technologies Used

- Python
- Pandas
- NumPy
- Plotly
- Dash
- Scikit-learn
- Matplotlib
- Seaborn

## Project Structure

```bash
bi-dashboard/
├── data/
├── notebooks/
├── src/
├── assets/
├── app.py
├── requirements.txt
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
```

## Run Dashboard

```bash
python app.py
```
